# Web Ollama Model Manager

The AI & Providers page now supports:
- downloading an Ollama model by name
- live download progress
- listing installed local models
- selecting the active local model
- testing a model
- removing a model
- opening the official Ollama model library

All Ollama API traffic remains server-side and uses:
http://127.0.0.1:11434

Port 11434 should not be exposed publicly.
